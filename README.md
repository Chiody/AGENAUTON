## 快速开始

```bash
git clone https://github.com/Chiody/AGENAUTON.git
cd agenauton

# 打开官网
open index.html

# 或打开在线引导
open Agenauton/guide/index.html

# 或复制 AI 指令到你的项目
cp Agenauton/03_Soul_Forging/ai_architect_prompt.md your_project/
```
